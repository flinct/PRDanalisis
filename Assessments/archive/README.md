# Assessment Archive

Folder ini menyimpan dokumen analisis historis yang dipindahkan dari struktur lama.

## Isi utama

- `legacy-temp-analysis/` — dokumen yang sebelumnya berada di `Temp Analysis/`

## Catatan

- Dokumen di sini dipertahankan untuk jejak historis.
- Dokumen archive belum tentu sudah mengikuti template assessment versi terbaru.
- Jika salah satu dokumen archive masih aktif dipakai sebagai referensi keputusan, promosikan ke struktur aktif:
  - `Assessments/<domain>/<feature-slug>/<feature-slug>-qa-assessment.md`
  - `Assessments/<domain>/<feature-slug>/versions/`
