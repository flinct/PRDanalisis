# Satuinbox Feature List 2

1. Autentikasi dan onboarding: login, register, verifikasi email/member, reset password, onboarding organisasi, upload NIB/KTP/NPWP, approval company.
2. Omnichannel inbox: WhatsApp, WhatsApp API, Email, Instagram, Messenger, livechat widget, dan group chat.
3. Conversation management: assign/unassign, close/reopen, star, pin, spam/junk, read/unread, bulk action, tag, notes, history, dan event timeline.
4. Chat room: kirim pesan, attachment/media, emoji, reply, edit/delete/copy/pin message, retry message, screenshot, dan typing indicator.
5. Contact management: list contact, tambah/edit contact, duplicate check, chat history, area sales/operational, dan send message.
6. Team inbox: grouping channel, member assignment, icon/color, dan routing conversation.
7. Agent/member management: invite member, role, shift, status active/away, max conversation, dan accept chat outside working hours.
8. Role dan permission: RBAC, masking email/phone, dan permission per menu/fitur.
9. Ticketing/helpdesk: create ticket, ticket type, stage, status, priority, assignee/team, custom fields, dan linked conversation/message.
10. Ticket action: resolve, snooze, tag, bulk reply, export, reminder, SLA ticket, dan shipment tracking.
11. Broadcast/campaign: broadcast WhatsApp, bulk upload, draft, schedule, template, variable, retry, dan analytics broadcast.
12. Broadcast template: create/edit/delete template, submit review WhatsApp API/Meta, language/category/header/body variables.
13. Analytics/reporting: conversation analytics, ticket analytics, responsiveness, member performance, CSAT, broadcast report, dan offline report/export.
14. SLA dan responsiveness: FRT, RLT/TTC, SLA per channel/conversation/ticket stage, reminder breach, dan office hours handling.
15. CSAT: survey customer untuk conversation/ticket, public CSAT link, setting per channel, dan response analytics.
16. Livechat widget: widget public, greeting/home, topic/subtopic, custom form, chat room, attachment, emoji, resume session, dan transcript.
17. Widget settings: branding, theme, launcher, logo, sound, whitelabel, email transcript, dan install script.
18. Channel integration: WhatsApp Web QR/session, WhatsApp API OAuth/embedded signup, Email IMAP/SMTP, Instagram, Facebook Messenger, dan Telegram add-on.
19. API/Open API: API key, partner Open API, request logging, webhook, dan correlation ID.
20. Webhook/developer tools: ticket status webhook, livechat transcript webhook, endpoint test, dan activate/deactivate webhook.
21. Notification center: in-app notification, unread/read, filter/search, billing/wallet/conversation/ticket notification, dan realtime socket.
22. Billing/subscription: plan/package, add-ons, voucher, billing cycle, invoice, payment, auto-renewal, downgrade, dan cancel.
23. Wallet/token: top up token, token balance, history, quota usage, dan broadcast wallet deduction.
24. Sales/CRM leads: lead list/pipeline, create lead, change team inbox, comments, company data, dan contact data.
25. Visit management: create visit, visit history, approve/reject visit, duplicate warning, prospect, existing, dan rejected visit.
26. Organization settings: company profile, language, API key, dan general organization settings.
27. Shift dan working hours: default/member shift, working hour config, dan away reason.
28. Tags dan macros: tag active/archive dan macro message untuk chat.
29. Ticket workflow settings: ticket type, stage, custom attributes, dan SLA reminder.
30. Shipping credential: vendor/environment/status credential CRUD untuk shipment/tracking.
31. Media dan transcript: upload/stream/delete media, shared media token, dan public transcript token.
32. Audit/observability: Open API audit log, request/response status, duration, IP, user agent, dan API key identifier.

Catatan: beberapa bagian frontend terlihat masih placeholder atau parsial, termasuk route contact tertentu, payment details subscription, dan WhatsApp API setting page dengan dummy/stub.
