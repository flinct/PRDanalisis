# Scripts

Folder ini menyimpan helper script yang dipakai untuk analisis, parsing testcase, migration, dan automation support.

## Struktur

- `analysis/` — script aktif untuk analisis/parsing/migration
- `analysis/legacy-temp-analysis/` — script hasil migrasi dari struktur lama `Temp Analysis/scripts/`

## Aturan

- Jangan simpan script di root repo.
- Jika script masih relevan untuk workflow, simpan di `Scripts/analysis/`.
- Script historis yang belum dirapikan boleh tetap disimpan di `analysis/legacy-temp-analysis/` sampai nanti dinormalisasi.
